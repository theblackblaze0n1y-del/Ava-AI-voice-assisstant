# Ava AI Voice Assistant

A voice-enabled AI assistant built with Python.

## Features

- Speech Recognition
- Groq AI Integration
- Natural Female Voice
- Conversation Memory
- Speech Interruption
- Friendly AI Personality
- press I to interrupt

## Technologies

- Python
- Groq
- SpeechRecognition
- Edge-TTS
- Pygame

## Requirements

- python-3.10
- groq api 
- SpeechRecognition
- edge-tts
- pygame
- keyboard
- pyaudio(if Ava is running already then not require)
